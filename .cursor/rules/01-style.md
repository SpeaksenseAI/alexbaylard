# Style and Conventions

When unsure, say "Unknown"—point me to file/URL. Never invent APIs.

Prefer code over prose. Default to a single code block; add 3-5 bullets only if needed.

Reference sources: when using 3rd-party APIs, cite file paths or URLs.

Signature-first: before coding, list the 3-5 symbols you will touch (file.ts: fn()).

Types over comments: introduce/extend TypeScript types before implementation.
